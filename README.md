
Application en ligne de commande permettant de gérer des fichiers et dossiers depuis le terminal.

## Fonctionnalités

- Créer un fichier
- Supprimer un fichier
- Lister les fichiers
- Lire un fichier
- Renommer un fichier
- Ajouter du contenu à un fichier
- Vider un fichier
- Copier un fichier
- Rechercher du texte dans des fichiers
- Créer un dossier
- Supprimer un dossier

## Technologies utilisées

- Python
- os
- glob
- shutil

## Compétences travaillées

- Fonctions
- Boucles
- Conditions
- Manipulation de fichiers
- Manipulation de dossiers
- Modules Python
- Organisation du code
